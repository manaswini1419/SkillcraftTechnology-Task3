# SkillCraft Technology – Task 3 (Quiz Application)
This repository contains the submission for Task 3 of the SkillCraft Technology internship.
## Project: Quiz Web Application
A dynamic quiz application built using core web technologies.  
It allows users to answer multiple-choice questions and view their score at the end of the quiz.
## Features
- Multiple-choice questions
- Next / Previous navigation
- Final score calculation
- Submit quiz functionality
- Simple and user-friendly interface
## Technologies Used
- HTML
- CSS
- JavaScript
## Description
This project demonstrates the use of JavaScript for handling quiz logic, user interactions, and score calculation, along with HTML for structure and CSS for styling.
